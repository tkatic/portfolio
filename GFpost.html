Gamefruit Website

This was the final project in my first programming class CGT 14100, which was named, "Internet Foundations Technologies and Development."

This course taught me all of my foundations for web programming, focusing on HTML and CSS. I had also learned some basic Javascript to create button functions on the website.

My team and I decided to make a mock website that would have all of the functions and features of a gaming forum.

I was tasked with making the content, ideas, and desiging the layout for the website by coding most of the HTML and CSS.

This project was worked on for two months and while it is not my best work, it was a fun learning experience that drove me to learn and make more programs and websites.

Technologies:
  • - HTML/CSS
  • - Javascript
  • - Adobe Photoshop
  • - Adobe Illustrator
