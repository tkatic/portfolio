Lands' End Performance Dashboard

This was the main project that I was assigned during my internship at Lands' End. I was tasked with making a dashboard that would interact with the PageSpeed Insights API and render the data according to filters.

The project began with a CLI application in Node JS that sent an HTTP request to the PageSpeed Insights API based on a few parameters, such as url.

I then built a REST API that automated the request and stored the data into MongoDB. The final part of the project was to build the webpage with an interactive dashboard and useful filters such as url, metric, and date.

The best part of this project was that the tool was set to be used by roughly 500 internal employees. It felt amazing to develop something by myself that would be used and improve the efficiency of their analytics.

I worked on this project for 3 months during the duration of my internship as a Web Engineer at Lands' End.

Technologies:
  • - Javascript
  • - Node JS
  • - PageSpeed API
  • - MongoDB
  • - HTML/CSS
  • - Docker
